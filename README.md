# DMTOS 23.05  
This branch contains the current profile for building the `iso` for DMTOS 23.05.
# Build
To build this branch, you need an Arch system and `archiso`

Use this command to build DMTOS 23.05 iso:
```
sudo mkarchiso -v -w . -o ./out/ .
```
